Why this module?

Currently Odoo does not have approval process customer creation.
So this module will help you to solve this problem.

Features

  • Have separate menu as 'Draft Customer' for newly created customers.

  • Approval process for Administrator user for customer record.

  • Separate filter as 'Approved Customer' to identify approved Customers.

Screenshots

1) Create your new customer using 'Create & Edit' option


2) You can see your newly created customer record in 'Draft Customers' menu.


3) Administrator user can approve the customer since we used 'Settings' group in this button.You can use any group as per your need.


4) Added separate filter to identify 'Approved Customer'
